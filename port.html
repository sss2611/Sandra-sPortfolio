Hola, soy Sandra Soledad Sanchez

Sobre mí

Técnica en Desarrollo de Software especializada en aplicaciones web y móviles, análisis estadístico de datos y automatización inteligente.

Soy una apasionada del desarrollo de software con enfoque en la experiencia del usuario y la eficiencia del código. Me gusta aprender nuevas tecnologías y trabajar en proyectos que generen impacto.

Me motiva transformar información en soluciones útiles, integrando tecnologías modernas como Python, Pandas, Jupyter Notebook y herramientas de visualización para generar insights reales.

Participo activamente en proyectos institucionales, académicos y colaborativos que combinan diseño, funcionalidad y escalabilidad, siempre con una visión integral de producto y experiencia de usuario.

¿Sabías que…?

Además de programar, me encanta resolver problemas complejos y crear soluciones intuitivas para los usuarios.

Tuve la oportunidad, gracias al ITSE, de participar en el Rally de Innovación 2025, una experiencia que fortaleció mi creatividad, trabajo en equipo y habilidades técnicas en entornos de alto desafío.

Proyectos destacados

Mi primer portfolio

Desarrollado con HTML, CSS y JavaScript. Incluye formulario de login y mensajes con SweetAlert2, además de Bootstrap para el diseño. Desplegado en GitHub Pages.

Veterinaria Cuky

Landing page informativa creada solo con HTML y CSS. Desplegada en GitHub Pages.

Proyectos con Vue, Flask y Django

  • NiñeraxHora: Aplicación web con Vue.js para control de empleadas domésticas.

Muestra de proyecto para Patrimonio

Proyecto presentado al director de Patrimonio en Prácticas Profesionalizantes I. Incluye login y validaciones con HTML, CSS y JavaScript. Desplegado en GitHub Pages.

FirstP

Proyecto personal con HTML, CSS, Bootstrap y JavaScript. Incluye login y dashboard para carga de productos. Permite cambiar el nombre de la marca y estilos con Bootswatch. Base de datos en MongoDB Atlas e imágenes alojadas en Cloudinary.

Actualmente funciona como e-commerce orientado a ventas por WhatsApp para clientes de Santiago del Estero y La Banda. Al ser escalable, se proyecta integrar una API de pagos dentro del sitio, con soporte para tarjetas de crédito/débito, Mercado Pago, Go Cuotas y otras pasarelas digitales.

EsTodoMadera

E-commerce donde el carrito redirecciona a WhatsApp. Desarrollado con tecnologías web y desplegado en Render.

Festival Sonidos de la Ciudad - Proyecto académico

Trabajo práctico de Programación 3. Página web con HTML, CSS y JavaScript desplegada en GitHub Pages. Posteriormente convertida en aplicación móvil con Flutter y Dart. Disponible para descarga desde Google Drive.

Sonidos SGO - Reto académico

Migré la página web del Festival Sonidos de la Ciudad a una aplicación móvil utilizando Flutter y Dart. Disponible para descargala desde mi Google Drive.

ID-Cultural

Proyecto en desarrollo para la Subsecretaría de Cultura de la provincia, en el marco de Prácticas Profesionalizantes III. Utilizamos una máquina virtual con Ubuntu, Docker para contenerización, y tecnologías como PHP, MySQL, Bootstrap con Bootswatch, CSS y JavaScript. El objetivo es crear una plataforma cultural funcional y escalable que responda a necesidades reales del sector público.
