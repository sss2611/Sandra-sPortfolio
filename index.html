Hola, soy Sandra Soledad Sanchez

Técnica en Desarrollo de Software especializada en aplicaciones web y móviles, con experiencia en scraping de páginas, análisis estadístico de datos y automatización inteligente.

Me apasiona transformar información en soluciones útiles, integrando tecnologías modernas como Python, Pandas, Jupyter Notebook y herramientas de visualización para generar insights reales. También participo en proyectos institucionales, académicos y colaborativos que combinan diseño, funcionalidad y escalabilidad.

Muestra de proyecto para Patrimonio

FirstP

Proyecto personal con HTML, CSS, Bootstrap y JavaScript. Incluye login y dashboard para carga de productos. Permite cambiar el nombre de la marca y estilos con Bootswatch. Base de datos en MongoDB Atlas e imágenes alojadas en Cloudinary.

Actualmente funciona como e-commerce orientado a ventas por WhatsApp para clientes de Santiago del Estero y La Banda. Al ser escalable, se proyecta integrar una API de pagos dentro del sitio, con soporte para tarjetas de crédito/débito, Mercado Pago, Go Cuotas y otras pasarelas digitales.

EsTodoMadera

ID-Cultural

Proyecto en desarrollo para la Subsecretaría de Cultura de la provincia, en el marco de Prácticas Profesionalizantes III. Utilizamos una máquina virtual con Ubuntu, Docker para contenerización, y tecnologías como PHP, MySQL, Bootstrap con Bootswatch, CSS y JavaScript. El objetivo es crear una plataforma cultural funcional y escalable que responda a necesidades reales del sector público.
